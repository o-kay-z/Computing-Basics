Software

Software controls the information and operation in the computer. It consists of programs (Application Software/System Software) that allow a computer to carry out specific tasks, distinguishing it from the physical parts of the system, known as hardware. To ensure the smooth operation of a data processing system, it includes together a set of instructions, procedures, algorithms, and related documentation. Software makes it possible for users to accomplish a wide range of tasks, from basic calculations to complex data analysis.

Application Software

Used to solve a particular problem or to perform a particular task, may either be custom-designed or packaged. Real word tasks such as keeping track of inventory, designing a car engine, drafting minutes for meeting, and playing games are possible for users because of it.

System Software

Manages and supports the operation of a computer system, often interacting directly with hardware. It includes operating systems, hardware drivers, and programming languages, which are typically not of concern to regular end-users.

Programming Languages
Types of Programming Languages Description
Procedural Programming Language execute a sequence of statements using variables, loops, and other elements, often controlling variables and performing tasks like printing information, unlike functional programming languages.
Functional Programming Language focus on return values and avoid loops, using recursive functions instead, while discouraging side effects and state storage; they are more abstract but can be harder to understand at the machine level, making them less suited for system programming.
Object-oriented Programming Language Object-oriented programming languages view the world as objects with internal data and external access, emphasizing encapsulation, inheritance for reusability, and polymorphism to extend implementations without extensive code changes.
Scripting Programming Language typically procedural with some object-oriented elements, but they are not considered full-fledged programming languages for developing large systems.
Logic Programming Language Declarative languages allow programmers to make statements about what should be done, leaving the machine to determine how to achieve the result by imposing constraints on what it must consider.

Implementation in Communities

Education

Computer-Based Education (CBE) is implemented in order to allow institutions and teachers control, deliver, and evaluate the student’s learning. It helps institutions track and analyze student performance.

Healthcare

Used for patient record-keeping, diagnostics, and medical procedures, including monitoring systems, lab diagnostics, and surgeries. (Electronic Health Record Software)

Communication

Computers are used for Email, chatting, Usenet, FTP, Telnet, and video conferencing.

Insurance

Insurance companies use computers to maintain up-to-date records of clients, such as start dates, due installments, maturity dates, interest, survival benefits, and bonuses. (AMS/PMS)

Engineering

Computer-Aided Design (CAD) used in various engineering fields, including structural engineering for stress and strain analysis, industrial engineering for system design and improvement, and architectural engineering for town planning and building design using 2D and 3D drawings.

Government

Computers are used for budgeting, sales and income tax departments, gender ratio calculations, voter list management, PAN card issuance, and weather forecasting. (PMS)

Military

Computers are heavily used in defense for controlling modern tanks, missiles, weapons, and military systems. Examples: missile control, military communication, operation and planning, and smart weapons communication.
